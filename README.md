# CrowdFundingWebsite
Telerik Academy Node.js team project

## Prerequisites

Globally installed 
- eslint, babel-eslint, eslint-config-google
- yarn
- gulp

## Getting started
1. Restore dependencies with command `yarn`
2. Start up development envirnoment with `gulp dev`
